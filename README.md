# vivienda-segmentacion-shiny
App web en R Shiny para explorar y analizar beneficiarios. Filtra por sexo, provincia/cantón, segmento, discapacidad (CONADIS) y rango de ingresos. Incluye gráficos interactivos, estadísticas interpretadas, semáforo financiero (déficit y ratio gasto/ingreso), rankings y descarga CSV.
